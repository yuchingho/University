Basic Web-App Principles

This simple demo app contains all of the features you would need to provide in a web-app (excepting object-orientation, which has been thoroughly covered elsewhere). In it you will find HTML mark-up and related code for a simple User-Interface and the event handlers needed to drive it.

Below you can see a simple user-interface - a text box with buttons to collect the text from the text-box into Javascript variables, and to clear the current text-box content. Text added using this U-I will be appended to a list which will be displayed in the bottom box.

For the U-I elements to be usable from Javascript, each has been given an ID property. A good convention to use in this is that every ID should be lower-case text.

In the Javascript code, we will need to set up a number of things:

  • Storage for the values entered. This will be a simple Javascript array.
  • Functions to extract the text from the text box and assign it to Javascript variables, and to clear out the text from the text box.
  • Event Handlers - functions to react to user-interface events, such as the user clicking on a button

To set up the above list, we will use with 

window.onload
event handler that the browser provides. This will wire up the events we want to react to to the code we want executed in response to these events.
