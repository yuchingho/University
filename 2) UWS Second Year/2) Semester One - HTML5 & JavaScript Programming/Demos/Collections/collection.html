A Custom Collection type

The collection class embedded in this page contains a number of tests whose results can be viewed by opening the console window (Ctrl+Shift+J).

Once the console is open, you can try out the collection type for yourself.

Go through the following steps:

  1. c = new Collection();
  2. c.add('one', 1);
  3. c.add('two', 'TWO');
  4. c.add('fred', 'fred@bloggo.com');
  5. c.add('Google', 'www.google.co.uk');
  6. c.add('img', new Image());
  7. c.keys(); // should show the list of keys.
  8. f = function( obj ) { console.log( obj.toString(); }
  9. c.forEach(f); // should list all of the objects
  10. function Thing(){}; // A new type
  11. c = new Collection(Thing); // c is now a typed collection
  12. c.add("Fred", "Fred"); // should return false - "Fred" not a thing
  13. a.add("Thing", new Thing()); // should return true - a Thing
