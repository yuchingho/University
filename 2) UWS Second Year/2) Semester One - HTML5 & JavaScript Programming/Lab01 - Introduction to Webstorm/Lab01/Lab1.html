HTML 5 Test Page

Info about this project:

This project is intended as a "get started in HTML & Javascript" demonstration.

The activities in it are centered around building a simple HTML5 page and then getting it to interact with Javascript code.

Abilities you should expect to gain from completing this lab:

  • Creating a simple HTML file that loads properly into a browser.
  • Adding HTML controls for list selection, button input, date input and numerical range input
  • Attaching a Javascript code file to the HTML file
  • Adding HTML elements to a HTML file through scripting.
  • Triggering JS code using events - particularly window.onload, button.onclick, list.onchange and input.oninput.
  • Performing simple operations such a date arithmetic in Javascript.
